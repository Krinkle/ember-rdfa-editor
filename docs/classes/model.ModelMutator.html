Class ModelMutator<T>

A mutator is the only publicly accessible way to change the model. It is available through the Model.change method. It provides an interface on top of operations for use mainly inside commands.

This construction allows for a rich api with all the access it needs while still maintaining full encapsulation and enabling things like batch operation processing
