createIri

  • createIri<B>(term: string, activeTag: IActiveTag, vocab: boolean, allowSafeCurie: boolean, allowBlankNode: B): B extends true ? ModelBlankNode | ModelNamedNode<string> : ModelNamedNode<string>
    • Defined in addon/utils/rdfa-parser/util.ts:329

    Create a named node for the given term. This will take care of prefix detection.

    Type parameters

    • B: boolean

    Parameters

    • term: string

      A term string (CURIE or IRI, aka safe-CURIE in RDFa spec).

    • activeTag: IActiveTag

      The current active tag.

    • vocab: boolean

      If creating an IRI in vocab-mode (based on vocab IRI), or in base-mode (based on base IRI).

    • allowSafeCurie: boolean

      If safe CURIEs are allowed (invalid CURIEs between square brackets will return null) Otherwise, only IRIs are allowed.

    • allowBlankNode: B

      If blank nodes are allowed. Otherwise null will be returned.

    Returns B extends true ? ModelBlankNode | ModelNamedNode<string> : ModelNamedNode<string>

    An RDF term or null.
